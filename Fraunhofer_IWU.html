Predictive Modeling of Tool Life in Hard Turning Operations Using Advanced Machine Learning Algorithms

This Project was carried out under the guidance of Prof. Dr.-Ing. Martin Dix Director at Fraunhofer IWU,Chemnitz.This project was part of a publication led by Dr. Martin Dix for the CIRP Society.

In this project, I developed two distinct approaches to predict tool life in hard turning operations, utilizing both time-domain and frequency-domain analyses. The first approach focused on time-domain data, where I used moving averages and standard deviation differences of cutting forces to model tool behavior over time. These weighted features were normalized and scaled, forming the foundation for a robust predictive framework. I then implemented the K-Nearest Neighbors (KNN) algorithm, correlating force behavior with known tool failure times to accurately predict tool life. This method provided a clear, time-based understanding of the tool wear process and its progression.

The second approach analyzed the frequency domain using Fast Fourier Transform (FFT). By converting time-series data into frequency spectra, I extracted meaningful features that were binned into fixed-length vectors representing the amplitude spectrum. These features were fed into an Isolation Forest algorithm, designed to detect anomalies that could signify tool wear or failure with an accuracy of 86% of predicting the tool failure in the Penultimate cut. This frequency-based approach complemented the time-domain analysis by uncovering patterns and anomalies that were not evident in the time series. Together, these two methods demonstrated my ability to integrate signal processing and machine learning to solve real-world engineering problems, providing a comprehensive solution for tool life prediction.
