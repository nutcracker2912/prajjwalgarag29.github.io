Python Projects

I’m a Python developer who loves rolling up my sleeves and building everything from data-driven applications to intelligent automation solutions. Over the years, I’ve tackled API development, crunched numbers with Pandas and NumPy, and created eye-catching GUIs using TKinter. Thanks to my solid background in object-oriented programming, I can design code that’s both efficient and easy to maintain—plus, my cloud automation know-how lets me quickly deploy and manage apps in today’s ever-evolving cloud environments.

On the machine learning side, I’ve explored frameworks like TensorFlow, Keras, and scikit-learn, pairing them with matplotlib to bring data insights to life. My portfolio includes a mix of hands-on projects that reflect this variety: interactive OOP quiz apps that keep users engaged, NLP-powered workout trackers for personalized fitness insights, and real-time weather and stock trading apps packed with helpful analytics. I’ve even built some classic games (hello, Snake and Pong!) to keep things fun. Whether it’s crafting a simple GUI or diving into complex algorithms, I’m committed to creating high-quality, innovative solutions that genuinely help people and make a real impact.
