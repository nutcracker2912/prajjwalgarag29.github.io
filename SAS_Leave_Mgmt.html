SAS Employee Leave Management System

As a freelance developer I developed an Employee Leave and Duty Management System for an Oman based Company Seeh Al-Sarya to simplify the process of managing employee schedules across multiple locations. This system is built on a robust database that stores employee details, such as IDs, designations, work statuses, and leave periods, categorized by sites like Fahud-oil and Yibal-gas. The centralized database ensures that all information is organized and easily accessible, serving as the foundation for efficient employee management.

To keep the system up-to-date, I implemented an automation module that runs daily to update employee statuses based on leave dates. This module automatically transitions employees between "on leave" and "working" statuses and adjusts dates accordingly, additionally it sends a mail to the manager alerting of employees going or coming from leave. By eliminating the need for manual updates, this feature ensures that managers always have accurate, real-time information about their workforce, improving productivity and reducing errors for the middle level management.

The project also features an interactive graphical user interface (GUI) built with Tkinter, making it easy to use. The GUI allows users to search for employees, view their details, add or edit records, and generate timetables in a visually appealing layout. I added functionality to display on-duty employees in a card-style format, categorized by location, and included light/dark mode themes for a better user experience. This project highlights my ability to integrate automation, data handling, and user-friendly design into a cohesive solution tailored to streamline complex workflows.
